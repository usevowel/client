Voice Nag & Terms/Privacy Modal Example

✨ Voice Nag

A friendly promotional message that introduces users to the voice agent. Auto-dismisses on first use or manual acknowledgment.

📋 Terms & Privacy

Require users to accept terms and privacy policy before using the voice agent. Blocks usage until accepted.

💾 LocalStorage

All interactions are persisted to localStorage so users don't see the same prompts again.

<vowel-voice-widget app-id="your-app-id" preset="shopify" position="bottom-right" show-transcripts="true" enable-nag="true" nag-title="Try Voice Shopping! 🎤" nag-description="Get personalized help finding products, checking out, and more—all hands-free." enable-terms-modal="true" terms-url="https://example.com/terms" privacy-url="https://example.com/privacy" ></vowel-voice-widget>

🔍 How It Works

  1. First Visit: You'll see the voice nag message wrapping the mic button
  2. Click "Got it": The nag dismisses and won't show again
  3. Click Mic Button: If terms modal is enabled, you'll see the terms/privacy modal
  4. Accept Terms: Voice agent connects and terms acceptance is stored
  5. Reload Page: Neither prompt will show again (stored in localStorage)
