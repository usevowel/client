# Vowel for Shopify - Marketing Content

This directory contains source materials for marketing the Vowel Voice Assistant Shopify app.

## Purpose

These markdown files are **marketing content** for:
- Shopify App Store listing
- Landing page for Shopify merchants
- Feature documentation
- Sales materials
- Support documentation

## Structure

```
about/
├── README.md                  # This file
├── overview.md                # Shopify app overview and value proposition
├── features.md                # Key features for merchants
├── how-it-works.md            # How the integration works (merchant-focused)
├── installation.md            # Installation and setup guide
├── use-cases.md               # Shopify-specific use cases
├── pricing.md                 # Pricing for Shopify merchants
└── faq.md                     # Frequently asked questions
```

## Usage

1. **App Store Listing** - Use this content for Shopify App Store description
2. **Landing Page** - Transform into marketing landing page for Shopify merchants
3. **Support Docs** - Provide help documentation for app users
4. **Sales Materials** - Use in outreach to Shopify merchants

## Not Technical Documentation

This is **merchant-facing content**, not developer documentation:
- ✅ Benefits for store owners
- ✅ Customer experience improvements
- ✅ ROI and conversion benefits
- ❌ Code examples
- ❌ API documentation
- ❌ Developer implementation details

For technical content, see:
- `/client/shopify/app/README.md` - Developer documentation
- `/client/shopify/app/QUICK_START.md` - Quick start guide
- `/client/docs/` - General Vowel documentation
