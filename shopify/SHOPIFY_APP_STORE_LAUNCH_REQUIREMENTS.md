## 🔌 App Positioning: Connector to External Platform

**Important:** This app is positioned as a **connector/integration** to the Vowel platform, similar to apps like:
- Mailchimp for Shopify (email marketing connector)
- Google Analytics (analytics connector)
- Klaviyo (marketing automation connector)

**How It Works:**
1. Merchant creates account on vowel.to platform
2. Merchant configures Vowel app and gets App ID
3. Merchant pays Vowel directly for platform usage
4. Merchant installs Shopify app (free)
5. Merchant enters Vowel App ID + Gemini API key in Shopify app
6. Shopify app acts as thin client to Vowel platform

**Benefits of This Approach:**
- ✅ Simplified billing (external to Shopify)
- ✅ No data storage in Shopify app
- ✅ Simplified GDPR compliance
- ✅ Clear separation of concerns
- ✅ Reduced technical complexity
- ✅ Common, accepted pattern on App Store

# Privacy Policy - Vowel Voice Assistant for Shopify

Last Updated: January 2026

## Overview

Vowel Voice Assistant for Shopify is a **connector application** that 
integrates your Shopify store with the Vowel voice platform (vowel.to).

This app does NOT store, process, or retain customer data. All voice 
processing and data handling occurs on the Vowel platform.

## What This Connector App Does

1. Embeds Vowel voice widget in your Shopify store theme
2. Passes your Vowel App ID to the widget
3. Enables communication between your store and Vowel platform

## Data Collection by This Connector App

**We DO NOT collect or store:**
- ❌ Customer voice recordings
- ❌ Customer personal information
- ❌ Shopping cart contents
- ❌ Order information
- ❌ Payment information

**We ONLY store:**
- ✅ Your Vowel App ID (configuration)
- ✅ Widget settings (position, preferences)
- ✅ Installation timestamp

This configuration data is stored only in your Shopify store's 
app settings and is deleted when you uninstall the app.

## Data Processing by Vowel Platform

When customers use voice features, their data is processed by:

1. **Vowel Platform (vowel.to)**
   - See Vowel's privacy policy: https://vowel.to/privacy
   
2. **Google Gemini Live API (your API key)**
   - See Google's privacy policy: https://policies.google.com/privacy

**Important:** You (the merchant) control:
- Your Vowel platform account and subscription
- Your Google Gemini API key
- What voice features are enabled

## Merchant Responsibilities

As a merchant using this connector app, you are responsible for:

1. Creating and managing your Vowel platform account
2. Providing and securing your Gemini API key
3. Complying with Vowel's terms of service
4. Informing your customers about voice features
5. Compliance with privacy laws in your jurisdiction

## Customer Data Rights (GDPR/CCPA)

Since this connector app stores no customer data:

**For Data Requests:**
- This app has no customer data to provide
- Customer data is handled by Vowel platform
- Contact Vowel platform for data access requests

**For Data Deletion:**
- Uninstalling this app removes configuration only
- To delete customer voice data, contact Vowel platform
- Store configuration is automatically deleted on uninstall

## Third-Party Services

This connector app facilitates connections to:

**Vowel Platform (vowel.to)**
- Purpose: Voice AI processing and management
- Data handling: See Vowel's privacy policy
- Your relationship: Direct account and billing

**Google Gemini API (via your API key)**
- Purpose: Voice recognition and AI processing  
- Data handling: See Google's privacy policy
- Your relationship: You provide and control the API key

## Cookies and Tracking

This connector app does NOT use cookies or tracking.

The Vowel platform widget may use browser storage for:
- Session management (temporary)
- User preferences (optional)

See Vowel's privacy policy for details.

## Children's Privacy

This app is intended for merchant use only and does not target 
children. Merchants are responsible for compliance with COPPA 
and similar laws in their jurisdiction.

## Changes to This Policy

We will notify you of significant changes via email or app dashboard.
Continued use after changes constitutes acceptance.

- Email: shopify@vowel.to
- Website: https://vowel.to/shopify

- Email: privacy@vowel.to

**Summary:** This is a connector app with no data storage.

### 3.2 Billing (Connector App - External) ✅ ACCEPTABLE

**Connector App Billing Model:**
- ✅ **App is FREE to install** from Shopify App Store
- ✅ **External billing** through Vowel platform (vowel.to)
- ✅ **No Shopify Billing API required** (connector app exception)
- ✅ Merchant pays Vowel directly for platform usage

**This is a Standard Pattern:**
Similar to:
- Mailchimp (free connector, pay Mailchimp for email service)
- Google Analytics (free connector, pay Google for premium features)
- Zendesk (free connector, pay Zendesk for support platform)
- Klaviyo (free connector, pay Klaviyo for marketing automation)

**Shopify's Position on This:**
Shopify explicitly allows free connector apps that charge externally when:
1. The app connects to an external service/platform
2. The external service has its own account system
3. The app listing clearly discloses external charges required
4. The primary service is provided by external platform

**What We Must Disclose in Listing:**
```
Pricing: Free to install

Additional charges may apply:
- Vowel Platform subscription (starting at $0/month free tier)
- Google Gemini API usage (provided by merchant)

This app is a connector to the Vowel platform. You must create 
a Vowel account at vowel.to to use this app.
```

**Benefits of This Model:**
- ✅ No Shopify Billing API implementation needed
- ✅ Direct relationship with customers
- ✅ Flexible pricing outside Shopify's constraints
- ✅ Simplified app code
- ✅ Common, accepted pattern

**Status:** ✅ Acceptable - No Shopify Billing API needed

### 3.4 Compliance Webhooks (Simplified for Connector) ⚠️ NEEDS IMPLEMENTATION

**Required Webhooks for App Store:**

1. **customers/data_request** (GDPR)
   - Merchant requests customer data
   - Must respond with all data for that customer
   - 30 days to respond

2. **customers/redact** (GDPR)
   - Merchant requests customer data deletion
   - Must delete all customer data
   - 30 days to respond

3. **shop/redact** (GDPR)
   - Store requests data deletion after uninstall
   - Must delete all store data
   - 30 days to respond

**Simplified for Connector Apps:**

Since this is a connector app with NO data storage in the Shopify app itself:

**What Data We Store (in Shopify app context):**
- ❌ No customer voice recordings (processed by Vowel platform)
- ❌ No customer personal data (handled by Shopify)
- ❌ No transaction data (handled by Shopify)
- ✅ Only: Store configuration (Vowel App ID, settings)

**Webhook Implementation (Simplified):**

```javascript
// customers/data_request - Return empty or minimal data
{
  "customer_id": "123",
  "data": {
    "note": "This app is a connector to Vowel platform. No customer data stored.",
    "vowel_platform": "Data stored in Vowel platform at vowel.to"
  }
}

// customers/redact - Nothing to delete at app level
{
  "customer_id": "123",
  "redacted": true,
  "note": "No customer data stored in connector app. Deletion forwarded to Vowel platform."
}

// shop/redact - Delete store configuration
{
  "shop_id": "store.myshopify.com",
  "redacted": true,
  "note": "Store configuration deleted. Vowel platform notified."
}
```

**Implementation Needed:**
- Webhook endpoint: `https://api.vowel.to/webhooks/shopify/compliance`
- Verify webhook authenticity (HMAC)
- Log all requests for compliance audit
- Respond with appropriate JSON
- Forward deletion requests to Vowel platform API
- Actually delete store config from any cache/database

**Key Point:** Since we're a connector with no data storage, compliance is MUCH simpler. We just need to:
1. Acknowledge the request
2. Delete any store configuration we cache
3. Forward to Vowel platform for actual data handling
